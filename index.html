IPCONFIG

Affiche l'adresse IP, le masque de sous-réseau et la passerelle par défaut de chaque carte liée à TCP/IP.

OPTIONUTILISATIONEXEMPLE
/?Affiche un message d'aideipconfig /?
/allAffiche toutes les informations de configurationipconfig /all
/releaseLibère l'adresse IPv4 pour la carte spécifiéeipconfig /release
/release6Libère l'adresse IPv6 pour la carte spécifiéeipconfig /release6
/renewRenouvelle l'adresse IPv4 pour la carte spécifiéeipconfig /renew
/renew6Renouvelle l'adresse IPv6 pour la carte spécifiéeipconfig /renew6
/displaydnsAffiche le contenu du cache de résolution DNSipconfig /displaydns
/flushdnsVide le cache de résolution DNSipconfig /flushdns

PING

Teste la connectivité avec un hôte (nom DNS ou adresse IP).

OPTIONUTILISATIONEXEMPLE
-tEffectue un test ping sur l'hôte spécifié jusqu'à son arrêt. Pour arrêter, appuyez sur Ctrl+C.ping -t 10.0.0.1
-n countNombre de demandes d'écho à envoyerping -n 5 10.0.0.1
-l sizeTaille du tampon d'envoiping -l 100 10.0.0.1
-i TTLDurée de vieping -i 25 10.0.0.1
-w timeoutDélai d'attente pour chaque réponse, en millisecondesping -w 2000 10.0.0.1
-4Force l'utilisation d'IPv4ping -4 www.google.fr
-6Force l'utilisation d'IPv6ping -6 www.google.fr

TRACERT

Définit le chemin vers un hôte (nom DNS ou adresse IP).

OPTIONUTILISATIONEXEMPLE
-h SautsMaxiNombre maximum de sauts pour rechercher la cibletracert -h 10 8.8.8.8
-4Force l'utilisation d'IPv4tracert -4 www.google.fr
-6Force l'utilisation d'IPv6tracert -6 www.google.fr

ROUTE

Gère les tables de routage réseau.

OPTIONUTILISATIONEXEMPLE
printAffiche la table de routage de l'hôteroute print
addAjoute une route dans la table de routageroute add 10.0.0.0 MASK 255.0.0.0 10.0.0.255
deleteSupprime une route dans la table de routageroute delete 10.0.0.0
changechange la passerelle d'une route dans la table de routageroute change 10.0.0.0 MASK 255.0.0.0 10.255.255.255

ARP

Affiche et modifie les tables de traduction d'adresses IP en adresses physiques utilisées par le protocole de résolution d'adresses ARP.

OPTIONUTILISATIONEXEMPLE
-aAffiche les entrées ARP en coursarp -a
-dSupprime l'hôte spécifié par adresse IP. Le caractère générique * permet de supprimer tous les hôtesarp -d 10.0.0.1
-sAjoute l'hôte et associe l'adresse IP avec l'adresse MAC. L'entrée est permanentearp -s 10.0.0.1
